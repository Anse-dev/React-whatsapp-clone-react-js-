# Getting Started

This project is whatsapp web cloning

## dowload project

In terminal run:

### `git clone `

## Download and install all dependencies

### `npm install`

Great everything is ready to start the server

### `npm run start` or `yarn run start`

Bingo you can change the resources as you like.
